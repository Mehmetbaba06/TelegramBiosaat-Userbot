## Yapılandırma Vars
1. `API_ID` : Telegram API_ID, my.telegram.org/apps
2. `API_HASH` : Telegram API_ID, my.telegram.org/apps'dan alın
3. `SESSION_NAME` : Geçerli bir Pyrogram oturum adı
4. `TIME_ZONE`: Saat diliminiz (Örnek: Turkey/İstanbul)
